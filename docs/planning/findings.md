# Findings

## Research
- **Ollama API**: Supports `POST /api/generate` and `/api/chat`.
- **Structured Outputs**: Can define JSON schema for predictable test case formats.
- **Libraries**: `ollama-python` is the recommended Python client.
- **Models**: `llama3`, `mistral`, `codellama` are suitable for code tasks.
- **Approaches**: Use few-shot prompting to guide the LLM to generate structured test cases from source code.
